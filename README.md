This is my little corner of the internet to share my projects, experience, and the kind of stuff I like building.

## About

I wanted this to feel clean, beautiful, and a bit more alive than a standard portfolio site, so it mixes minimal layouts and smooth motion without getting in the way of the content.

It is also intentionally overengineered. This is a `pnpm` + Turbo monorepo for one app, which is objectively unnecessary and also exactly the point.
